# 🌦️ Simple & Modern Weather App

A sleek, responsive, and modern weather application that provides real-time weather information for any city around the world. Built with a clean UI and smooth user experience, this app helps users quickly access current weather conditions, temperature, humidity, wind speed, and other essential weather details. The project follows modern web design principles and offers a fast, intuitive interface for everyday use.

## 🚀 Live Demo

🔗 **Live Website:** https://apm4u.github.io/weatherapp/

## ✨ Features

* 🌍 Search weather by city name
* 🌡️ Real-time temperature updates
* 💧 Humidity information
* 🌬️ Wind speed details
* ☁️ Weather condition display
* 📱 Fully responsive design
* 🎨 Modern and clean user interface
* ⚡ Fast and lightweight performance
* 🔍 Easy-to-use search functionality

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* Weather API
* Responsive Web Design

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/apm4u/weatherapp.git
```

2. Navigate to the project folder

```bash
cd weatherapp
```

3. Open `index.html` in your browser

## 🔑 API Configuration

If you're using a weather API, add your API key in the JavaScript file:

```javascript
const apiKey = "YOUR_API_KEY";
```
